Pixel Interactive Piece 11/24

Posted by Azaraus on Monday, July 7th 2025

Use your phone for the inverted filter

This is progress for my CVL final. In this project, I am using HTML, CSS, and JavaScript (p5.js) to create an interactive pixel art piece. Users can interact with the piece by changing the fill color of the pixels.

How Did I Do This?

I first used p5.js to create my pixel art piece. I started with using a video capture to display the user on the webcam I then came up with a formula to manipulate the pixels based on their position and color values. Finally, I added interactivity by creating buttons that allow users to change the fill color of the pixels.

I had to come up with a way for the button to randomly change the fill color. I did this by creating a custom variable that generates colors using different values After that I used that variable in the fill() function to change the color of the pixels when the button is pressed.

After that I wanted my header to match the fill color of the person on the screen. So I used JavaScript to target the h2 element in my HTML and then added what I wanted to happen to that variable/h2 element. I then added the action I wanted to happen which was the hover function. Finally I set the color of the h2 to the the variable I made to change the fill color of the pixels.

You can learn more at p5.js.

Hopefully you learned something new here. See you next time!
